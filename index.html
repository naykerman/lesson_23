Оператори

Варіант №1 не вірний, так як в нас два вираза і оператор "+" не перетворює тип String на Number, для вирішення потрібно забрати "+".

Варіант №3 не вірний так як інкремент використовується зі змінними, а 22 число, для виправлення потрібно помістити 22++ в лапки.

Варіант №4 не вірний, змінній newUsers присвоюється початкове значення usersCounter через суфіксну форму інкременту, для вирішення потрібно поміняти форму інкременту на префіксну.

Варіант №5 не вірний, вернеться перший вираз, так як там обидва вираза з оператором && True і оператор || вертає тільки перше true яке знаходить.

Варіант №6 Верне 0 так як оператор ?? вертає перший аргумент, якщо він не null або undefined, в іншому випадку буде виводитись "без імені"

Умовне вітвлення

Варіант №1 виведе "Брехня!", так як оператор === не обробляє типи і вирази.

Варіант №2 виведе "Правда!", так як оператор == завжди пробує перетворити типи.

Варіант №3 виведе "Правда!", так як умова буде True.

Варіант №4 виведе 'Правда!', так як умова в if верне False.

Цикли

Задача №2 цикл не верне 0, так як цикл виводить тільки значення True, а 0 буде False.

Рядки

Задача №1 значення змінної потрібно взяти в зворотні лапки.

Задача №3 не вірна так як оператор + не вміє переводити рядки в числа і додавання виконуватись не буде.

Задача №6 значення буде false так як 'лан' починається на третьому елементі.
